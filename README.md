# vagrant-sandbox
A PHP application development environment with Vagrant.

## Getting Started
```bash
git clone git@github.com:dit-rohm/vagrant-sandbox.git
vagrant up
```

* /public_html : http://localhost:8080/
* phpMyAdmin : http://localhost:8080/phpmyadmin
